.. _dijit/_BidiSupport:

==================
dijit._BidiSupport
==================

*Deprecated*.

This module used to be used to enable bidi support in dijit, but it's deprecated.

Bidi support for dijit should now be enabled via:

.. html ::

  <script src="dojo.js" data-dojo-config="has:{'dojo-bidi': true}"></script>
