Summary¶

Reports all has features.

Description¶

Finds and reports all has feature tests encountered in any module included in a build.

Profile Knobs¶

hasReport (default = 0)
  • [falsy] No report is output.
  • [truthy] Output report.

Source Location¶

util/build/transforms/hasReport.js
